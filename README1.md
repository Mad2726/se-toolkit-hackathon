# Screenshot Placeholders

This directory should contain the following screenshots:

1. **start.png** - Bot start screen showing the welcome message and START button
2. **course.png** - Course selection menu (B23, B24, B25, M25)
3. **gpa.png** - GPA input prompt
4. **result.png** - Scholarship calculation result

## How to Add Screenshots

1. Run the bot in Telegram
2. Take screenshots of each step in the conversation
3. Save them in this directory with the names above
4. Recommended size: 800x600 or similar
5. PNG format preferred

## Temporary Placeholder

Until real screenshots are added, you can create simple placeholder images using any image editor or online tool.
